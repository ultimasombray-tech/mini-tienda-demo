This project is a **mini online store demo** created with **HTML, CSS, and JavaScript**. It is designed to showcase a fully functional example of a small online store that can be customized for clients.

## Features
- **Modern and responsive design** (works on both mobile and desktop).  
- **WhatsApp integration button** for easy customer contact or purchases.  
- **Customizable layout**: change colors, products, text, and images to fit your business.  
- **Ready-to-use demo**: perfect for presenting to clients before customizing the final version.  
- **Interactive elements**: buttons, hover effects, and basic animations included.  
- **Easy to modify**: all HTML, CSS, and JS files are organized for quick updates.

## How to Use
1. Open `index.html` in any web browser.  
2. Explore the demo and test functionality.  
3. Customize by editing the files in `css/`, `js/`, or `images/`.
